iPlantGeo

This is a prototype demonstration (milestone 0) of the provision of web services for accessing advanced data including environmental GIS layers, phylogenies, traits and gene frequencies using locations (latitude/longitude) and species binomials (Genus species).

Instructions

Start with either lat/lon or species binomials. In the matching box paste the requested information or specify a CSV (comma separated) file containing that information. NO HEADER ROW on the CSV input file. Then click the OK button.
